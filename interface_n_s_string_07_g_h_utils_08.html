- (NSArray *) gh_substringSegmentsWithinStart: (NSString *)  start
end: (NSString *)  end 

Break string into segments based on start and end token.

Parameters:
start Start token
end End token
Returns:
Array of GHNSStringSegment's

Use a regex engine if you can. Note: This exists because regex.h is posix only and does not support non-greedy expressions. Why Apple must you not give us objc regex library?

Get string segments, within start and end tokens. For example, 

    [@"This is <START>a test<END> string" subStringSegmentsWithinStart:@"<START>" end:@"<END>"] => [@"This is ", @"a test", @" string"]
