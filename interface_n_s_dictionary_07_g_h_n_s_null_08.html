+ (id) gh_dictionaryWithKeysAndObjectsMaybeNil: (id)  firstObject
,   NS_REQUIRES_NIL_TERMINATION 

Create dictionary which supports nil values.

Key is first (instead of value then key). If the value is nil it is stored internally as NSNull, and when calling objectMaybeNilForKey will return nil.

For example, [NSDictionary gh_dictionaryWithKeysAndObjectsMaybeNil:"key1", nil, @"key2", @"value2", @"key3", nil, nil];

Parameters:
firstObject... Alternating key, value pairs. Terminated when _key_ is nil.
